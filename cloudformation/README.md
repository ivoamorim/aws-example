CloudFormation Example
=========

Simple tutorial provision several services with CloudFormation.

Requirements
------------

- ansible
- boto3

How To Use
------------

Validate Template
------------

```
$ npm install -g cfn-lint
$ cfn-lint validate template.yml
0 infos
0 warn
0 crit
Template valid!
```
